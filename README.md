# Fashion-MNIST CNN Implementation
Fashion-MNIST Classification using Optimized CNNsThis project implements a state-of-the-art Deep Learning model for the classification of fashion article images, based on the research from the Indian Institute of Technology Patna.  👕 Project OverviewThe goal of this project is to classify images from the Fashion-MNIST dataset—a collection of Zalando's fashion article images containing 70,000 examples (60,000 training and 10,000 testing). Each example is a 28x28 grayscale image associated with one of 10 fashion classes.  The 10 Fashion Classes:T-Shirt/TopTrouserPulloverDressCoatSandalsShirtSneakerBagAnkle Boots   🧠 Model ArchitecturesThis repository compares three different Convolutional Neural Network (CNN) architectures to evaluate the impact of modern deep learning techniques:  Vanilla CNN2: A baseline model featuring two convolutional layers and max-pooling layers.  CNN2 + BatchNorm: Incorporates Batch Normalization before every convolutional layer. This technique normalized inputs to have zero mean and unit variance, enabling the model to reach the same loss in 10 epochs that previously took 40.  CNN2 + BatchNorm + Skip (SOTA): The final optimized model utilizing Residual Skip Connections. By adding the previous input directly to the convoluted output , the model ensures gradient signals propagate effectively during training.  ⚙️ Training ParametersAll models utilize a standardized backend for fair comparison:MLP Hidden Layer: 128 neurons with ReLU activation.  Regularization: 50% Dropout to prevent overfitting and improve generalization.  Optimizer: Adam optimizer for loss function minimization.  Output: 10 neurons with Softmax activation for probability distribution across the 10 classes.  📊 ResultsThe proposed optimized model shows impressive results, outperforming existing systems in literature:  ModelTest Accuracy (%)SVC (Baseline)89.70%   EDEN (Baseline)90.60%   CNN2 + BatchNorm + Skip92.54%   🔍 Key Insights from ResearchVisualization: Lower layers detect sharp features (boundaries/corners), while specific kernels act as "sleeve detectors" or "strap detectors".  Error Analysis: The primary source of error is the confusion between T-Shirts, Pullovers, Coats, and Shirts due to the visual similarity in 28x28 grayscale images.  References:
